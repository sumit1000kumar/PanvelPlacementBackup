# Setup Instructions

## Backend
1. Go to backend folder
2. Run: npm install
3. Run: node index.js
4. Ensure it's running at: http://localhost:5000

## Frontend
1. Open admin-job.html in VS Code
2. Right-click and select "Open with Live Server"
3. Page will open at: http://127.0.0.1:5500/admin-job.html
